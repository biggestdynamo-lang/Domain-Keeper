# Freeable Domains Platform - Complete Replit Agent Specification

Build a production-ready full-stack platform called "Freeable Domains".

The platform is a combination of:

* Domain Registrar
* Domain Registry
* DNS Manager
* GitHub Import Hosting Platform
* Environment Variable Manager
* Deployment Platform
* Infrastructure Manager
* Serverless Hosting Platform
* Website Builder
* Project Manager

The platform should be inspired by:

* Vercel
* Netlify
* Railway
* Render
* Cloudflare Pages
* GitHub
* Namecheap
* Porkbun
* Freenom (legacy concept only)
* InfinityFree
* Firebase Hosting

The entire platform must be completely self-hosted and built using modern TypeScript technologies.
